## Local Setup
* Ensure that your Mongo server is running by entering the following in command line:
    1. `mongod`
* In (pantryraid) project folder, enter the following commands:
    1. `npm install`
    1. `cd client`
    1. `npm install`
    1. `cd ..`
    1. `npm run seed`
    1. `npm start`
